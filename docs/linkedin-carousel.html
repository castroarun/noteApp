How to Save as PDF

  1. Press Ctrl+P (or Cmd+P on Mac)
  2. Select "Save as PDF" as the destination
  3. Set margins to "None"
  4. Enable "Background graphics"
  5. Save and upload to LinkedIn as a Document post
