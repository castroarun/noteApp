# Note App - UI Prototypes & Navigation Flow

**Version:** 2.0
**Date:** 2025-11-16
**Purpose:** Visual prototypes with applied color palette

---

## Design Principle

**Immediate Note-Taking:** Start typing the moment you open the app. Center focus on content creation with collapsible side panels for organization.

**Jira Auto-Detection:** App intelligently detects Jira task structure as you type. No separate page needed.

---
